## [0.1.0] - 2026-06-11

### 新增

#### 核心功能
- 实时行情监控，支持自选股分组管理
- 14 阶段精细化调度器
- 6 数据源容灾机制（东方财富、腾讯、同花顺、新浪、大智慧、基金）
- SQLite 数据库 + WAL 模式

#### 行情功能
- K 线图表（日K/周K/月K）
- 分时走势图 + 五日分时
- 技术指标（MA、EMA、MACD、BOLL）
- 复权切换（前复权/后复权/不复权）

#### A 股增强
- 6 板块差异化涨跌停引擎
- 创业板 20% 历史切换（2020.8.24 前为 10%）
- 新股首日临停规则
- 封板强度计算（排除午休）
- 异动检测（快速拉升/下跌/量比突增/封板炸板）

#### 持仓管理
- 持仓盈亏计算
- 组合收益汇总
- 超额收益对比（沪深300 基准）

#### 预警系统
- 11 种预警规则类型
- 系统原生 Toast 通知

#### 系统功能
- 系统托盘（跨平台适配）
- 开机自启
- 全局快捷键
- 悬浮窗

#### 市场数据
- 涨跌分布
- 热门股票
- 涨停天梯
- 板块行情

### 技术架构
- Tauri 2.x + Vue 3 + TypeScript
- Rust 后端 + SQLite 数据库
- Pinia 状态管理
- Naive UI 组件库
- LightweightCharts + ECharts 图表
