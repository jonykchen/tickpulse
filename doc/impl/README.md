## 核心模块一览

### Rust 后端模块 (src-tauri/src/)

| 模块 | 文件 | 功能 |
|------|------|------|
| `market/` | scheduler.rs + 6数据源 | 14阶段调度 + 多源容灾 |
| `analysis/` | engine + dimensions + llm | 7维度分析 + 双LLM架构 |
| `db/` | migrations + CRUD | SQLite + WAL模式 |
| `alert/` | rules + trigger | 预警规则引擎 |
| `anomaly/` | detector | 异动检测（涨速/量比） |
| `limit/` | board_type | 涨跌停判断（6板块） |
| `indicator/` | MA/EMA/MACD/BOLL | 技术指标计算 |
| `system/` | tray + hotkey + autostart | 系统托盘/快捷键/自启 |
| `health/` | diagnostics | 健康诊断面板 |
| `datasource/` | circuit_breaker + rate_limiter | 断路器 + 限流器 |
| `sidecar/` | bridge | Python IPC桥接 |
| `sync/` | client | 云同步客户端 |
| `logging/` | buffer + persistence | 日志系统 |
