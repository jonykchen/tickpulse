# S05 — 行情调度器
> 阶段：Phase 1 | 前置依赖：S03, S04 | 来源：v2.0 §3.4

## 1. 概述

行情调度器负责根据交易阶段动态调整刷新频率，实现 14 阶段精细化调度。支持交易所差异化收盘阶段、静默期优化、数据自愈、增量推送、异动检测、预警检查等功能。

## 4. 核心逻辑

### 4.1 MarketPhase::current() 方法（含交易所差异化）

```rust
impl MarketPhase {
    /// v2.0: 增加交易所参数，区分收盘阶段
    pub fn current(is_trading_day: bool, exchange: Exchange) -> Self {
        if !is_trading_day { return Self::Holiday; }
        let now = Local::now();
        let (h, m) = (now.hour(), now.minute());
        match (h, m) {
            (0..=8, _) | (9, 0..=14) => Self::PreMarket,
            (9, 15..=19) => Self::AuctionCancelable,
            (9, 20..=24) => Self::AuctionUncancelable,
            (9, 25..=29) => Self::PreOpen,
            (9, 30..=59) | (10, 0..=30) => Self::MorningVolatile,
            (10, 31..=59) | (11, 0..=30) => Self::MorningStable,
            (11, 31..=59) | (12, _) => Self::LunchBreak,
            (13, 0..=30) => Self::AfternoonOpen,
            (13, 31..=59) | (14, 0..=56) => Self::AfternoonStable,
            (14, 57..=59) => {
                // v2.0: 沪市主板无收盘竞价，仍为连续竞价
                match exchange {
                    Exchange::SZSE => Self::ClosingAuction,       // 深交所收盘竞价
                    Exchange::SSE => Self::ContinuousClosing,      // 沪市主板连续竞价
                    Exchange::BSE => Self::ClosingAuction,         // 北交所收盘竞价
                    _ => Self::ClosingAuction,
                }
            },
            (15, 0..=4) => Self::AfterHours,
            (15, 5..=30) => {
                // v2.0: 科创板/创业板有盘后固定价格交易
                match exchange {
                    Exchange::SSE | Exchange::SZSE => Self::PostMarketTrading,
                    _ => Self::PostMarketEnd,
                }
            },
            _ => Self::PostMarketEnd,
        }
    }
```

### 4.5 MarketScheduler::run() 主循环（含数据自愈）

```rust
impl MarketScheduler {
    pub async fn run(&self) {
        let mut consecutive_errors = 0u32;

        loop {
            let is_trading_day = self.trade_calendar.is_trading_day().await;
            // v2.0: 取主要交易所（深交所）确定全局阶段
            let phase = MarketPhase::current(is_trading_day, Exchange::SZSE);
            *self.current_phase.write().await = phase;

            self.app_handle.emit("scheduler-status", SchedulerStatus {
                phase: phase.display_name().to_string(),
                interval_secs: phase.refresh_interval().as_secs(),
                is_trading_day,
            }).ok();

            let interval = phase.refresh_interval();
            if phase.has_realtime_data() {
                match self.refresh_and_emit().await {
                    Ok(()) => consecutive_errors = 0,
                    Err(e) => {
                        consecutive_errors += 1;
                        tracing::error!("行情刷新连续失败 #{}: {}", consecutive_errors, e);
                        if consecutive_errors >= 10 {
                            tracing::warn!("连续失败10次，重建HTTP客户端");
                            self.rebuild_http_client().await;
                            consecutive_errors = 0;
                        }
                        tokio::time::sleep(
                            Duration::from_secs(5 * consecutive_errors.min(6) as u64)
                        ).await;
                        continue;
                    }
                }
            }

            tokio::select! {
                _ = tokio::time::sleep(interval) => {},
                _ = self.cancel_token.cancelled() => { break; }
            }
        }
    }
```

### 4.6 refresh_and_emit() 方法（增量推送 + 异动检测 + 预警检查）

```rust
    /// 增量行情推送 + 异动检测 + 预警检查
    /// v2.0: 静默期(PreOpen)无变化时跳过emit
    async fn refresh_and_emit(&self) -> Result<()> {
        let quotes = self.fetch_watchlist_quotes().await?;

        // 增量过滤
        let changed: Vec<_> = quotes.iter()
            .filter(|q| self.prev_quotes.get(&q.secid)
                .map_or(true, |p| p.price != q.price || p.change_percent != p.change_percent))
            .cloned()
            .collect();

        // v2.0: 静默期(9:25-9:30)无变化则跳过推送，减少无效IPC
        let current_phase = *self.current_phase.read().await;
        if current_phase == MarketPhase::PreOpen && changed.is_empty() {
            // 静默期无变化，跳过emit
        } else {
            // 异动检测
            let anomalies = self.anomaly_detector.lock().await.detect(&quotes, &self.prev_quotes);
            if !anomalies.is_empty() {
                self.app_handle.emit("anomaly-detected", &anomalies).ok();
            }

            // 预警检查
            self.alert_manager.lock().await.check_and_notify(&quotes);

            // 更新快照 + 推送
            for q in &quotes { self.prev_quotes.insert(q.secid.clone(), q.clone()); }
            if !changed.is_empty() { self.app_handle.emit("stock-update", &changed).ok(); }

            // 涨跌家数摘要
            let summary = MarketSummary::from_quotes(&quotes);
            self.app_handle.emit("market-summary", &summary).ok();

            // 托盘更新
            if let Some(first) = quotes.first() { self.update_tray_text(first).await; }
        }
        Ok(())
    }
}
```

## 5. 测试要点

1. **阶段切换**：验证 14 个阶段在正确时间点切换
2. **交易所差异**：验证沪市主板 ContinuousClosing、深交所 ClosingAuction
3. **静默期优化**：验证 PreOpen 阶段无变化时跳过 emit
4. **连续错误处理**：验证 10 次连续失败后重建 HTTP 客户端
5. **指数退避**：验证错误重试间隔 5s → 10s → 15s → 20s → 25s → 30s
6. **增量推送**：验证仅推送价格或涨跌幅变化的股票
7. **异动检测**：验证快速拉升/下跌/量比突增/封板炸板/临停检测
8. **预警检查**：验证预警规则触发和系统 Toast 通知
