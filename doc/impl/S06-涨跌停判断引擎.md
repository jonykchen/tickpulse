# S06 — 涨跌停判断引擎
> 阶段：Phase 2 | 前置依赖：S03(行情调度器) | 来源：v2.0 §3.5 + §3.6

## 1. 概述

涨跌停判断引擎负责 A 股各板块涨跌停价格计算、股票状态识别、封板强度计算以及新股首日临时停牌规则。v2.0 关键修正：**涨停价向上取整、跌停价向下取整**（非四舍五入），**创业板历史涨跌幅切换**（2020.8.24 前 10% / 后 20%），**封板率排除午休时间**。

核心职责：
- 板块类型识别（6 种 BoardType）
- 股票状态检测（4 种 StockStatus，正则匹配全角星号）
- 涨跌停价格计算（向上/向下取整到 0.01 元）
- 封板强度计算（有效交易时长 240 分钟，排除午休）
- 新股首日临时停牌规则（沪市/科创/创业差异化）

## 4. 核心逻辑

### 4.1 BoardType::from_secid — 板块识别

```rust
impl BoardType {
    pub fn from_secid(secid: &str) -> Self {
        let (_, code) = secid.split_once('.').unwrap_or(("0", ""));
        if secid.starts_with("116.") || secid.starts_with("134.") || secid.starts_with("2.") { return Self::HK; }
        if secid.starts_with("105.") || secid.starts_with("106.") { return Self::US; }
        match code {
            c if c.starts_with("688") => Self::STAR,
            c if c.starts_with("300") || c.starts_with("301") => Self::ChiNext,
            c if c.starts_with("8") || c.starts_with("4") => Self::BSE,
            _ => Self::MainBoard,
        }
    }
```

### 4.2 BoardType::limit_rate — 涨跌幅比例（含历史切换）

```rust
    /// v2.0: 增加date参数，创业板2020.8.24前为10%
    pub fn limit_rate(&self, status: StockStatus, date: NaiveDate) -> Option<Decimal> {
        match status {
            StockStatus::NewStock => None,  // 无限制
            StockStatus::ST => Some(Decimal::from_str("0.05").unwrap()),
            StockStatus::Delisting => Some(Decimal::from_str("0.10").unwrap()),
            StockStatus::Normal => match self {
                Self::MainBoard => Some(Decimal::from_str("0.10").unwrap()),
                Self::ChiNext => {
                    // v2.0: 创业板注册制改革日 2020-08-24
                    let reform_date = NaiveDate::from_ymd_opt(2020, 8, 24).unwrap();
                    if date < reform_date {
                        Some(Decimal::from_str("0.10").unwrap())  // 改革前10%
                    } else {
                        Some(Decimal::from_str("0.20").unwrap())  // 改革后20%
                    }
                },
                Self::STAR => Some(Decimal::from_str("0.20").unwrap()),
                Self::BSE => Some(Decimal::from_str("0.30").unwrap()),
                Self::HK | Self::US => None,
            },
        }
    }

    /// 判断是否为科创板或创业板（用于盘后交易和临停规则）
    pub fn has_post_market_trading(&self) -> bool {
        matches!(self, Self::STAR | Self::ChiNext)
    }
}
```

### 4.3 detect_stock_status — 股票状态检测

```rust
/// v2.0: 股票状态检测增强 — 正则匹配全角星号、退市前缀
pub fn detect_stock_status(name: &str, days_since_ipo: u32) -> StockStatus {
    if days_since_ipo <= 5 { return StockStatus::NewStock; }
    // 正则匹配：ST/*ST/S*ST/S＊ST/退市/退
    // 全角星号＊(U+FF0A) 和半角星号*(U+002A) 都匹配
    static ST_PATTERN: once_cell::sync::Lazy<regex::Regex> =
        once_cell::sync::Lazy::new(|| regex::Regex::new(r"(?i)(S?[＊*]?ST|退市|退)").unwrap());
    if ST_PATTERN.is_match(name) {
        if name.contains("退") { return StockStatus::Delisting; }
        return StockStatus::ST;
    }
    StockStatus::Normal
}
```

### 4.4 LimitCalculator::calculate — 涨跌停价格计算

```rust
/// 涨跌停计算 — v2.0: 涨停向上取整，跌停向下取整
pub struct LimitCalculator;

impl LimitCalculator {
    pub fn calculate(
        prev_close: Decimal,
        board: BoardType,
        status: StockStatus,
        date: NaiveDate,  // v2.0: 历史日期参数
    ) -> LimitResult {
        match board.limit_rate(status, date) {
            None => LimitResult { limit_up: None, limit_down: None, is_no_limit: true },
            Some(rate) => {
                // v2.0: 涨停价向上取整到0.01元，跌停价向下取整到0.01元
                // 例：5.67 × 1.05 = 5.9535 → 涨停价5.96（向上取整）
                // 例：5.67 × 0.95 = 5.3865 → 跌停价5.38（向下取整）
                let limit_up = (prev_close * (Decimal::ONE + rate))
                    .round_dp_with_strategy(2, RoundingStrategy::RoundUp);
                let limit_down = (prev_close * (Decimal::ONE - rate))
                    .round_dp_with_strategy(2, RoundingStrategy::RoundDown);
                LimitResult {
                    limit_up: Some(limit_up),
                    limit_down: Some(limit_down),
                    is_no_limit: false,
                }
            },
        }
    }
}
```

### 4.5 SealInfo::calc_seal_strength — 封板强度计算

```rust
/// v2.0: 封板率计算 — 排除午休时间
impl SealInfo {
    /// 有效交易时长 = 4小时 = 240分钟（排除午休11:30-13:00的90分钟）
    pub const EFFECTIVE_TRADING_MINUTES: f64 = 240.0;

    pub fn calc_seal_strength(seal_minutes: f64) -> f64 {
        (seal_minutes / Self::EFFECTIVE_TRADING_MINUTES).clamp(0.0, 1.0)
    }
}
```

### 4.6 TempSuspendRule::for_board — 新股临停规则

```rust
impl TempSuspendRule {
    pub fn for_board(board: BoardType) -> Option<Self> {
        match board {
            // 沪市主板新股：较开盘价 ±10% 停牌30分钟，±20% 停牌至14:57
            BoardType::MainBoard => Some(Self {
                board_type: board,
                thresholds: vec![
                    Decimal::from_str("0.10").unwrap(),
                    Decimal::from_str("0.20").unwrap(),
                ],
                suspend_minutes: vec![30, 0],  // 0=停牌至14:57
            }),
            // 科创板/创业板新股：较开盘价 ±30%/±60% 各停牌10分钟
            BoardType::STAR | BoardType::ChiNext => Some(Self {
                board_type: board,
                thresholds: vec![
                    Decimal::from_str("0.30").unwrap(),
                    Decimal::from_str("0.60").unwrap(),
                ],
                suspend_minutes: vec![10, 10],
            }),
            _ => None,
        }
    }
}
```

### 4.8 涨跌停取整规则示例

```
涨停价 = ⌈前收盘 × (1 + 涨跌幅)⌉ 向上取整到0.01元
跌停价 = ⌊前收盘 × (1 - 涨跌幅)⌋ 向下取整到0.01元

示例：前收盘 5.67，ST 涨跌幅 5%
  涨停价 = ⌈5.67 × 1.05⌉ = ⌈5.9535⌉ = 5.96 元
  跌停价 = ⌊5.67 × 0.95⌋ = ⌊5.3865⌋ = 5.38 元
```
