# S07 — 异动检测引擎
> 阶段：Phase 2 | 前置依赖：S06(涨跌停判断引擎) | 来源：v2.0 §3.7

## 1. 概述

异动检测引擎负责实时监控自选股行情数据，识别 A 股短线核心异动信号。v2.0 新增 `TempSuspend` 临停异动类型和量比衰减提示。

核心职责：
- 5 分钟价格快照与快速拉升/下跌检测（3% 阈值）
- 封板/炸板状态转换检测
- 新股临停状态转换检测
- 量比突增检测（从 <2 突增到 >5）
- 异动事件生成与推送

## 3. 数据结构

### 3.1 AnomalyDetector — 异动检测器

```rust
use std::collections::VecDeque;

pub struct AnomalyDetector {
    /// 每只股票最近5分钟价格快照
    price_history: DashMap<String, VecDeque<(i64, Decimal)>>,
    /// 量比历史
    volume_ratio_prev: DashMap<String, Decimal>,
    /// 涨停状态历史（检测封板/炸板）
    limit_up_prev: DashMap<String, bool>,
    /// 新股临停状态历史
    temp_suspend_prev: DashMap<String, bool>,
}
```

### 3.2 AnomalyType — 异动类型（6 种）

```rust
#[derive(Debug, Clone, Serialize)]
pub enum AnomalyType {
    RapidRise { period_secs: u64, change_pct: Decimal },    // 5分钟内涨幅>3%
    RapidFall { period_secs: u64, change_pct: Decimal },    // 5分钟内跌幅>3%
    VolumeSurge { prev: Decimal, curr: Decimal },           // 量比从<2突增到>5
    SealForm,   // 封板（非涨停→涨停）
    SealBreak,  // 炸板（涨停→非涨停）
    TempSuspend { reason: String, resume_time: Option<i64> },  // v2.0: 新股临停
}
```

### 3.3 AnomalyEvent — 异动事件

```rust
#[derive(Debug, Clone, Serialize)]
pub struct AnomalyEvent {
    pub secid: String,
    pub stock_name: String,
    pub anomaly_type: AnomalyType,
    pub timestamp: i64,
}
```

## 4. 核心逻辑

### 4.1 detect — 完整检测实现

```rust
impl AnomalyDetector {
    pub fn detect(
        &mut self,
        quotes: &[StockQuote],
        prev: &DashMap<String, StockQuote>,
    ) -> Vec<AnomalyEvent> {
        let now = chrono::Utc::now().timestamp();
        let mut events = Vec::new();

        for q in quotes {
            // 更新价格历史
            let history = self.price_history.entry(q.secid.clone()).or_default();
            history.push_back((now, q.price));
            // 保留最近5分钟
            while history.front().map_or(false, |(t, _)| now - *t > 300) {
                history.pop_front();
            }

            // 快速拉升/下跌检测
            if let Some(&(_, old_price)) = history.front() {
                if old_price > Decimal::ZERO {
                    let change = (q.price - old_price) / old_price * Decimal::from(100);
                    if change >= Decimal::from(3) {
                        events.push(AnomalyEvent {
                            secid: q.secid.clone(), stock_name: q.name.clone(),
                            anomaly_type: AnomalyType::RapidRise {
                                period_secs: 300, change_pct: change,
                            },
                            timestamp: now,
                        });
                    } else if change <= Decimal::from(-3) {
                        events.push(AnomalyEvent {
                            secid: q.secid.clone(), stock_name: q.name.clone(),
                            anomaly_type: AnomalyType::RapidFall {
                                period_secs: 300, change_pct: change,
                            },
                            timestamp: now,
                        });
                    }
                }
            }

            // 封板/炸板检测
            let was_limit_up = self.limit_up_prev.get(&q.secid)
                .map(|v| *v).unwrap_or(false);
            if q.is_limit_up && !was_limit_up {
                events.push(AnomalyEvent {
                    secid: q.secid.clone(), stock_name: q.name.clone(),
                    anomaly_type: AnomalyType::SealForm, timestamp: now,
                });
            } else if !q.is_limit_up && was_limit_up {
                events.push(AnomalyEvent {
                    secid: q.secid.clone(), stock_name: q.name.clone(),
                    anomaly_type: AnomalyType::SealBreak, timestamp: now,
                });
            }
            self.limit_up_prev.insert(q.secid.clone(), q.is_limit_up);

            // v2.0: 新股临停检测
            if q.stock_status == StockStatus::NewStock {
                let was_suspended = self.temp_suspend_prev.get(&q.secid)
                    .map(|v| *v).unwrap_or(false);
                if q.is_temp_suspended && !was_suspended {
                    events.push(AnomalyEvent {
                        secid: q.secid.clone(), stock_name: q.name.clone(),
                        anomaly_type: AnomalyType::TempSuspend {
                            reason: q.temp_suspend_reason.clone().unwrap_or_default(),
                            resume_time: q.temp_suspend_resume_time,
                        },
                        timestamp: now,
                    });
                }
                self.temp_suspend_prev.insert(q.secid.clone(), q.is_temp_suspended);
            }

            // 量比突增检测
            if let Some(vr) = q.volume_ratio {
                let prev_vr = self.volume_ratio_prev.get(&q.secid)
                    .map(|v| *v).unwrap_or(Decimal::ONE);
                if prev_vr < Decimal::from(2) && vr > Decimal::from(5) {
                    events.push(AnomalyEvent {
                        secid: q.secid.clone(), stock_name: q.name.clone(),
                        anomaly_type: AnomalyType::VolumeSurge { prev: prev_vr, curr: vr },
                        timestamp: now,
                    });
                }
                self.volume_ratio_prev.insert(q.secid.clone(), vr);
            }
        }
        events
    }
}
```

### 4.2 检测规则汇总

| 异动类型 | 触发条件 | 字段 |
|----------|----------|------|
| RapidRise | 5 分钟内涨幅 >= 3% | `period_secs: 300, change_pct` |
| RapidFall | 5 分钟内跌幅 <= -3% | `period_secs: 300, change_pct` |
| VolumeSurge | 量比从 <2 突增到 >5 | `prev, curr` |
| SealForm | 非涨停 → 涨停 | — |
| SealBreak | 涨停 → 非涨停 | — |
| TempSuspend | 新股且非临停 → 临停 | `reason, resume_time` |

## 5. 测试要点

| 测试场景 | 预期结果 |
|----------|----------|
| 5 分钟内涨幅 3.5% | 触发 RapidRise { period_secs: 300, change_pct: 3.5 } |
| 5 分钟内跌幅 -4.2% | 触发 RapidFall { period_secs: 300, change_pct: -4.2 } |
| 5 分钟内涨幅 2.9% | 不触发 |
| 量比从 1.5 突增到 6.0 | 触发 VolumeSurge { prev: 1.5, curr: 6.0 } |
| 量比从 2.5 增到 6.0 | 不触发（prev >= 2） |
| 量比从 1.0 增到 4.0 | 不触发（curr <= 5） |
| 非涨停 → 涨停 | 触发 SealForm |
| 涨停 → 非涨停 | 触发 SealBreak |
| 连续涨停 | 仅首次触发 SealForm |
| 新股首次临停 | 触发 TempSuspend { reason, resume_time } |
| 新股持续临停 | 仅首次触发 |
| 非新股临停 | 不触发 TempSuspend |
| 价格快照超过 5 分钟自动清理 | VecDeque 长度不无限增长 |
| 多只股票同时异动 | 返回多个 AnomalyEvent |
| 无异动 | 返回空 Vec |
