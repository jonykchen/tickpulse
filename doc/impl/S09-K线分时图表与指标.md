# S09 — K线分时图表与指标
> 阶段：Phase 2 | 前置依赖：S08(复权因子与K线缓存) | 来源：v2.0 §3.11 + §4.2 + 附录A

## 1. 概述

本模块负责 K 线/分时/五日分时图表类型定义、K 线面板完整组件结构、时间轴同步实现、分时均价线计算、K 线周期映射以及技术指标模块引用。

核心职责：
- 图表类型枚举（K线/分时/五日分时）与复权方式定义
- K 线面板组件树（工具栏 + 主图 + 副图 + 状态栏）
- K 线时间轴同步（多图表联动）
- 分时图均价线计算（累计成交金额/累计成交量）
- K 线周期与东方财富 API code 映射
- 技术指标模块（MA/MACD/BOLL/EMA）引用

### 4.2 useKlineChartSync — K线时间轴同步实现

```typescript
// composables/useKlineChart.ts
export function useKlineChartSync() {
  const charts = ref<IChartApi[]>([]);

  function registerChart(chart: IChartApi) {
    charts.value.push(chart);
  }

  function syncTimeScales() {
    const master = charts.value[0];
    if (!master) return;

    master.timeScale().subscribeVisibleLogicalRangeChange((range) => {
      if (!range) return;
      charts.value.slice(1).forEach((chart) => {
        chart.timeScale().setVisibleLogicalRange(range);
      });
    });
  }

  onMounted(() => {
    nextTick(() => syncTimeScales());
  });

  return { registerChart };
}
```

### 4.3 calcAvgPrice — 分时图均价线计算

```typescript
// composables/useTimelineChart.ts
/**
 * 均价线（黄线）= 累计成交金额 / 累计成交量
 * 这是A股分时图的核心要素，反映当日平均成交成本
 */
function calcAvgPrice(timelineData: TimelinePoint[]): TimelinePoint[] {
  let cumAmount = 0;  // 累计成交金额
  let cumVolume = 0;  // 累计成交量

  return timelineData.map((point) => {
    cumAmount += point.price * point.volume;
    cumVolume += point.volume;
    return {
      time: point.time,
      value: cumVolume > 0 ? cumAmount / cumVolume : point.price,
    };
  });
}
```
