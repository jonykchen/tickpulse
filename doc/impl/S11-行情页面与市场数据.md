## 3. 数据结构

### 3.1 BlockTrade — 大宗交易

v2.0 新增：大宗交易是 A 股重要信号源，折价率反映机构态度。

```rust
#[derive(Debug, Clone, Serialize)]
pub struct BlockTrade {
    pub secid: String,
    pub stock_name: String,
    pub trade_date: NaiveDate,
    pub trade_price: Decimal,           // 成交价
    pub trade_volume: i64,              // 成交量（股）
    pub trade_amount: Decimal,          // 成交金额（万元）
    pub premium_rate: Decimal,          // 折价率 = (成交价-收盘价)/收盘价×100%，负数=折价
    pub buyer_seat: Option<String>,     // 买方营业部
    pub seller_seat: Option<String>,    // 卖方营业部
}
```

### 4.4 大宗交易 API

东方财富大宗交易端点：

```
datacenter-web.eastmoney.com/api/data/v1/get?reportName=RPT_DABLOCKTRADE_DETAILS
```

`premium_rate`（折价率）计算逻辑：

```
折价率 = (成交价 - 收盘价) / 收盘价 × 100%
负数 = 折价成交（机构折价拿货，看空信号）
正数 = 溢价成交（机构溢价拿货，看多信号）
```

1. **大宗交易数据**：验证 BlockTrade 结构体字段正确解析，折价率计算正确（负数=折价）

   - MarketPhaseBar：各交易阶段文字正确显示
   - MarginTag："融" 标签蓝色小标签
   - VolumeRatioNote：开盘量比偏高警告
4. **自选股列配置**：14 列默认可见性正确（名称/状态/价格/走势图/总市值/成交量/当日涨跌/年初至今/备注 默认可见），可排序列正确
5. **行情页面组件**：各子组件数据加载正确，Tab 切换流畅
6. **大宗交易 Command**：验证 days 参数默认 30 天，返回 Vec<BlockTrade> 格式正确
