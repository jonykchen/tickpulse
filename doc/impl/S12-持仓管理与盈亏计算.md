# S12 — 持仓管理与盈亏计算

> 阶段：Phase 4 | 前置依赖：S03 | 来源：v2.0 §3.16 + §8

## 1. 概述

本模块负责持仓管理与盈亏计算，核心能力包括：

- **PortfolioSummary**：持仓汇总结构体，含总市值、浮动盈亏、当日盈亏、当日盈亏率、基准涨幅（沪深300）、超额收益
- **PositionCalculator**：完整的盈亏计算实现，所有计算使用 `rust_decimal` 确保金融级精度
- **超额收益**：持仓涨幅 vs 沪深300 基准，量化跑赢/跑输大盘程度

## 2. 接口定义

### 2.1 Tauri Commands

持仓 CRUD 命令（同原方案，此处不重复），盈亏计算在 Rust 侧完成，通过 `PortfolioSummary` 返回前端。

pub struct PortfolioSummary {

## 4. 核心逻辑

### 4.1 PositionCalculator — 完整实现

```rust
pub struct PositionCalculator;

impl PositionCalculator {
    pub fn market_value(quantity: Decimal, price: Decimal) -> Decimal { quantity * price }
    pub fn float_pnl(quantity: Decimal, price: Decimal, cost: Decimal) -> Decimal { (price - cost) * quantity }
    pub fn pnl_rate(price: Decimal, cost: Decimal) -> Decimal {
        if cost.is_zero() { return Decimal::ZERO; }
        ((price - cost) / cost * Decimal::from(100)).round_dp(2)
    }
    pub fn today_pnl(quantity: Decimal, price: Decimal, prev_close: Decimal) -> Decimal {
        (price - prev_close) * quantity
    }
    pub fn position_ratio(market_value: Decimal, total_value: Decimal) -> Decimal {
        if total_value.is_zero() { return Decimal::ZERO; }
        (market_value / total_value * Decimal::from(100)).round_dp(2)
    }

    pub fn summarize(
        positions: &[(Decimal, Decimal, Decimal, Decimal)],  // (qty, price, cost, prev_close)
        benchmark_change: Option<Decimal>,                     // 沪深300今日涨幅
    ) -> PortfolioSummary {
        let mut total_market = Decimal::ZERO;
        let mut total_float_pnl = Decimal::ZERO;
        let mut total_today_pnl = Decimal::ZERO;

        for &(qty, price, cost, prev_close) in positions {
            total_market += Self::market_value(qty, price);
            total_float_pnl += Self::float_pnl(qty, price, cost);
            total_today_pnl += Self::today_pnl(qty, price, prev_close);
        }

        let today_pnl_rate = if total_market.is_zero() { Decimal::ZERO }
            else { (total_today_pnl / total_market * Decimal::from(100)).round_dp(2) };

        let excess_return = benchmark_change.map(|bc| today_pnl_rate - bc);

        PortfolioSummary {
            total_market_value: total_market, total_float_pnl, total_today_pnl,
            today_pnl_rate, benchmark_change, excess_return,
        }
    }
}
```

### 4.2 计算公式速查

| 指标 | 公式 | 说明 |
|------|------|------|
| 市值 | `quantity * price` | 持仓市值 |
| 浮动盈亏 | `(price - cost) * quantity` | 相对成本价的盈亏 |
| 盈亏率 | `(price - cost) / cost * 100` | 浮动盈亏百分比，cost=0 时返回 0 |
| 当日盈亏 | `(price - prev_close) * quantity` | 相对昨收的盈亏 |
| 仓位占比 | `market_value / total_value * 100` | 单只股票占总仓位比例，total=0 时返回 0 |
| 当日盈亏率 | `total_today_pnl / total_market * 100` | 组合当日盈亏百分比 |
| 超额收益 | `today_pnl_rate - benchmark_change` | 组合涨幅 - 沪深300涨幅 |

## 5. 测试要点

1. **市值计算**：`market_value(100, 5.67)` = 567.00
2. **浮动盈亏**：`float_pnl(100, 6.00, 5.50)` = 50.00（盈利）；`float_pnl(100, 5.00, 5.50)` = -50.00（亏损）
3. **盈亏率**：`pnl_rate(6.00, 5.50)` = 9.09%；`pnl_rate(5.50, 0)` = 0（成本为零保护）
4. **当日盈亏**：`today_pnl(100, 6.00, 5.80)` = 20.00
5. **仓位占比**：`position_ratio(567, 5000)` = 11.34%；`position_ratio(567, 0)` = 0（总值为零保护）
6. **summarize 汇总**：多只持仓累加正确，当日盈亏率计算正确
7. **超额收益**：`today_pnl_rate=2.5, benchmark_change=2.0` → `excess_return=0.5`（跑赢）；`benchmark_change=None` → `excess_return=None`
8. **Decimal 精度**：所有计算使用 `rust_decimal`，无浮点误差
9. **round_dp(2)**：盈亏率和仓位占比保留两位小数
