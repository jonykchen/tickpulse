# S14 — A股交易规则完整增强

> 阶段：Phase 5 | 前置依赖：S03, S05, S07 | 来源：v2.0 §3.14 + §3.15

## 1. 概述

本模块覆盖 A 股交易规则的完整数据结构增强，核心包括：

- **StockQuote 完整数据结构**：v2.0 扩展后的全部字段，涵盖基础行情、换手率（流通/总）、三种市盈率、量比（含衰减标记）、涨跌停/停牌、新股临停、封板信息、融资融券、Level-2 预留
- **VolumeRatioNote 枚举**：量比衰减提示
- **量比计算公式**：含开盘 30 分钟偏高警告
- **换手率计算公式**：流通换手率 vs 总换手率
- **市盈率计算公式**：静态/动态/TTM 三种
- **MarketSummary**：涨跌家数结构体

## 3. 数据结构

### 3.1 StockQuote — 完整数据结构

v2.0 扩展：三种市盈率、换手率分母区分、量比衰减标记、融资融券、临停、大宗交易。

```rust
#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct StockQuote {
    // 基础字段
    pub secid: String, pub name: String,
    pub price: Decimal, pub change: Decimal, pub change_percent: Decimal,
    pub prev_close: Decimal, pub open: Decimal, pub high: Decimal, pub low: Decimal,
    pub volume: Decimal, pub turnover: Decimal,
    pub trade_date: NaiveDate,

    // 换手率 — v2.0: 区分流通换手率和总换手率
    pub turnover_rate: Option<Decimal>,         // 流通换手率（东方财富f8，常用）
    pub total_turnover_rate: Option<Decimal>,   // 总换手率（新增）

    // 市盈率 — v2.0: 三种PE
    pub pe_ttm: Option<Decimal>,               // TTM市盈率（最常用，默认展示）
    pub pe_dynamic: Option<Decimal>,           // 动态市盈率（东方财富f9）
    pub pe_static: Option<Decimal>,            // 静态市盈率（上年度EPS）

    // 量比 — v2.0: 衰减标记
    pub volume_ratio: Option<Decimal>,
    pub volume_ratio_note: Option<VolumeRatioNote>,  // 衰减提示

    pub total_market_cap: Option<Decimal>,
    pub ytd_change: Option<Decimal>, pub change_speed: Option<Decimal>,
    pub main_net_inflow: Option<Decimal>,

    // 涨跌停/停牌
    pub board_type: BoardType, pub stock_status: StockStatus,
    pub is_limit_up: bool, pub is_limit_down: bool, pub is_near_limit_up: bool,
    pub limit_up_price: Option<Decimal>, pub limit_down_price: Option<Decimal>,
    pub is_suspended: bool,

    // v2.0: 新股临停
    pub is_temp_suspended: bool,                         // 是否临时停牌
    pub temp_suspend_reason: Option<String>,             // 临停原因
    pub temp_suspend_resume_time: Option<i64>,           // 预计恢复时间

    // 封板信息
    pub seal_strength: Option<f64>,   // 0.0-1.0（排除午休的有效封板率）
    pub seal_break_count: u32,        // 炸板次数

    // v2.0: 融资融券
    pub is_margin_target: bool,                          // 是否为两融标的
    pub margin_balance: Option<Decimal>,                 // 融资余额（亿元）
    pub short_volume: Option<Decimal>,                   // 融券余量（万股）

    // Level-2 预留
    #[serde(skip_serializing_if = "Option::is_none")]
    pub bid_prices: Option<Vec<Decimal>>,
    #[serde(skip_serializing_if = "Option::is_none")]
    pub ask_prices: Option<Vec<Decimal>>,
    #[serde(skip_serializing_if = "Option::is_none")]
    pub bid_volumes: Option<Vec<i64>>,
    #[serde(skip_serializing_if = "Option::is_none")]
    pub ask_volumes: Option<Vec<i64>>,
}
```

**字段分组说明**：

| 分组 | 字段 | 说明 |
|------|------|------|
| 基础 | secid/name/price/change/change_percent/prev_close/open/high/low/volume/turnover/trade_date | 核心行情数据 |
| 换手率 | turnover_rate/total_turnover_rate | 流通换手率（f8常用）+ 总换手率（v2.0新增） |
| 市盈率 | pe_ttm/pe_dynamic/pe_static | TTM（默认展示）/ 动态（f9）/ 静态（f23） |
| 量比 | volume_ratio/volume_ratio_note | 量比 + 衰减提示（v2.0） |
| 其他指标 | total_market_cap/ytd_change/change_speed/main_net_inflow | 总市值/年初至今/涨速/主力净流入 |
| 涨跌停/停牌 | board_type/stock_status/is_limit_up/is_limit_down/is_near_limit_up/limit_up_price/limit_down_price/is_suspended | 涨跌停判断 + 停牌 |
| 新股临停 | is_temp_suspended/temp_suspend_reason/temp_suspend_resume_time | v2.0 新增 |
| 封板 | seal_strength/seal_break_count | 封板率（排除午休）+ 炸板次数 |
| 融资融券 | is_margin_target/margin_balance/short_volume | v2.0 新增 |
| Level-2 预留 | bid_prices/ask_prices/bid_volumes/ask_volumes | 五档买卖盘预留 |

## 4. 核心逻辑

### 4.1 量比计算公式

```
量比 = 现成交总手 / (5日每分钟均量 × 当日已开市分钟数)

5日每分钟均量 = 过去5个交易日总成交量 / (5 × 240分钟)

⚠️ 开盘前30分钟量比偏高（分母小），参考价值有限
→ volume_ratio_note = Some(VolumeRatioNote::Early)
→ 前端 VolumeRatioNote.vue 显示 "⚠开盘量比偏高"
```

**量比解读**：

| 量比范围 | 含义 |
|----------|------|
| < 0.8 | 缩量 |
| 0.8 - 1.5 | 正常 |
| 1.5 - 2.5 | 温和放量 |
| 2.5 - 5 | 明显放量 |
| > 5 | 剧烈放量 |

**开盘 30 分钟内量比偏高原因**：分母中"当日已开市分钟数"很小（如仅 5 分钟），导致量比虚高。此时设置 `volume_ratio_note = Some(VolumeRatioNote::Early)` 提醒用户。

### 4.2 换手率计算公式

```
流通换手率 = 成交量 / 流通股本 × 100%    （东方财富f8字段，A股常用）
总换手率 = 成交量 / 总股本 × 100%         （新增字段）
```

**区分说明**：

- **流通换手率**（`turnover_rate`）：分母为流通股本，东方财富 f8 字段，A 股常用指标
- **总换手率**（`total_turnover_rate`）：分母为总股本，v2.0 新增字段

对于有限售股的股票，流通换手率 > 总换手率。

### 4.3 市盈率计算公式

```
静态PE = 当前股价 / 上年度每股收益（年报已公布数据）
动态PE = 当前股价 / 预估当年每股收益（分析师预测，东方财富f9）
TTM PE = 当前股价 / 最近4个季度每股收益之和（最常用，默认展示）
```

**三种 PE 对比**：

| 类型 | 字段 | 数据来源 | 特点 |
|------|------|----------|------|
| 静态PE | `pe_static` | 上年度年报 EPS | 滞后但确定 |
| 动态PE | `pe_dynamic` | 分析师预测当年 EPS | 前瞻但不确定 |
| TTM PE | `pe_ttm` | 最近 4 季度 EPS 之和 | 平衡时效与确定性，默认展示 |

**东方财富字段映射**：
- `f9`：东方财富实际返回 TTM 市盈率（非动态，需验证）
- `f23`：静态市盈率

**前端展示**：通过 `AppConfig.peDisplayType` 配置默认展示哪种 PE（默认 `ttm`）。

### 4.4 MarketSummary 生成

```rust
impl MarketSummary {
    pub fn from_quotes(quotes: &[StockQuote]) -> Self {
        Self {
            up_count: quotes.iter().filter(|q| q.change_percent > Decimal::ZERO).count() as i32,
            down_count: quotes.iter().filter(|q| q.change_percent < Decimal::ZERO).count() as i32,
            flat_count: quotes.iter().filter(|q| q.change_percent == Decimal::ZERO).count() as i32,
            limit_up_count: quotes.iter().filter(|q| q.is_limit_up).count() as i32,
            limit_down_count: quotes.iter().filter(|q| q.is_limit_down).count() as i32,
        }
    }
}
```

MarketSummary 在每次行情刷新时计算，通过 `market-summary` 事件推送到前端，在 TabBar 中展示：

```
涨:1234 跌:890 平:56
```

## 5. 测试要点

1. **StockQuote 完整字段**：验证所有字段正确解析和序列化
2. **换手率区分**：`turnover_rate`（流通）vs `total_turnover_rate`（总），流通 >= 总
3. **三种 PE**：`pe_ttm`/`pe_dynamic`/`pe_static` 各自独立，默认展示 TTM
4. **量比衰减**：开盘 30 分钟内 `volume_ratio_note = Some(VolumeRatioNote::Early)`，30 分钟后为 None
5. **新股临停字段**：`is_temp_suspended`/`temp_suspend_reason`/`temp_suspend_resume_time` 正确填充
6. **封板信息**：`seal_strength` 范围 0.0-1.0，`seal_break_count` 正确计数
7. **融资融券**：`is_margin_target` 布尔判断，`margin_balance`/`short_volume` 可选
8. **Level-2 预留**：`bid_prices`/`ask_prices`/`bid_volumes`/`ask_volumes` 使用 `skip_serializing_if` 跳过 None
9. **MarketSummary**：涨跌家数计数正确，涨停/跌停家数正确
10. **量比公式**：验证 5 日每分钟均量计算，开盘 30 分钟内偏高标记
11. **换手率公式**：流通换手率 = 成交量/流通股本，总换手率 = 成交量/总股本
12. **市盈率公式**：静态/动态/TTM 三种计算方式正确
