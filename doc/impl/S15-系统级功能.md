# S15 — 系统级功能
> 阶段：Phase 6 | 前置依赖：S01 | 来源：v2.0 §9.1-9.4 + §4.7

## 1. 概述

本模块实现 Tauri 桌面应用的系统级功能，包括系统托盘、悬浮窗、全局快捷键、开机自启以及极简窗口配置。这些功能是桌面应用区别于 Web 应用的核心特性，提供全天候盯盘体验。

## 2. 接口定义

### 2.1 系统托盘接口

```rust
/// 更新托盘 Tooltip 文字
/// macOS 有字符长度限制，需截断处理
pub fn update_tray_text(tray: &TrayIcon, text: &str) {
    // macOS tooltip 长度限制 ~100字符
    #[cfg(target_os = "macos")]
    let display = &text[..text.len().min(80)];

    #[cfg(not(target_os = "macos"))]
    let display = text;

    tray.set_tooltip(Some(display)).ok();
}
```

```rust
/// 打开悬浮窗（极简窗口）
pub fn open_suspend_window(app: &tauri::AppHandle) -> Result<(), Box<dyn Error>> {
    let window = WebviewWindowBuilder::new(
        app, "suspend", WebviewUrl::App("/#/suspend".into()),
    )
    .title("盯盘")
    .inner_size(220.0, 300.0)
    .decorations(false)          // 无边框
    .always_on_top(true)         // 置顶
    .transparent(true)           // 透明背景
    .resizable(true)
    .skip_taskbar(true)          // 不显示在任务栏
    .build()?;

    // 支持拖拽移动
    window.listen("drag-move", move |event| {
        // 前端发送拖拽事件，后端移动窗口位置
    });

    Ok(())
}
```

### 2.3 全局快捷键接口

```rust
/// 注册全局快捷键
pub fn register_hotkeys(app: &tauri::AppHandle) {
    use tauri_plugin_global_shortcut::ShortcutState;

    // CmdOrCtrl+Shift+S 切换主窗口显示/隐藏
    app.global_shortcut().on_shortcut("CmdOrCtrl+Shift+S", move |app, _shortcut, event| {
        if event.state == ShortcutState::Pressed { toggle_main_window(app); }
    }).ok();

    // CmdOrCtrl+Shift+D 打开悬浮窗
    app.global_shortcut().on_shortcut("CmdOrCtrl+Shift+D", move |app, _shortcut, event| {
        if event.state == ShortcutState::Pressed { open_suspend_window(app); }
    }).ok();
}
```

## 3. 数据结构

### 3.1 极简窗口配置（SuspendConfig）

```typescript
interface SuspendConfig {
  setting: {
    opacity: number;       // 透明度 10-100，步进5，默认100
    colorNum: number;      // 色彩度 10-100，步进5，默认100
    bgColor: string;       // 背景色，默认 "#16181c"（黑）；可选白 "#FAF3F2" 或自定义HSV
    showEnum: boolean;     // 底部菜单显隐，默认 true
    refreshRate: 4 | 6 | 8; // 刷新间隔（秒），默认 6
    sortType: 'updown' | 'time'; // 排序方式，默认 updown（涨幅排序）
    showWechat: boolean;   // 微信面板，默认 false
  };
  group: {
    show: boolean;         // 分组面板显隐，默认 false
    type: 'optional' | 'position'; // 分组类型，默认 optional（自选）
    index: number;         // 当前分组索引，默认 0
  };
}
```

### 3.2 默认配置

```typescript
const DEFAULT_SUSPEND_CONFIG: SuspendConfig = {
  setting: {
    opacity: 100, colorNum: 100, bgColor: '#16181c',
    showEnum: true, refreshRate: 6, sortType: 'updown', showWechat: false,
  },
  group: { show: false, index: 0, type: 'optional' },
};
```

### 3.3 配置合并函数（含旧数据兼容）

```typescript
function mergeSuspendConfig(saved: Partial<SuspendConfig> | null): SuspendConfig {
  const config = { ...DEFAULT_SUSPEND_CONFIG };
  config.setting = saved?.setting && typeof saved.setting === 'object'
    ? { ...DEFAULT_SUSPEND_CONFIG.setting, ...saved.setting }
    : { ...DEFAULT_SUSPEND_CONFIG.setting };
  config.group = saved?.group && typeof saved.group === 'object'
    ? { ...DEFAULT_SUSPEND_CONFIG.group, ...saved.group }
    : { ...DEFAULT_SUSPEND_CONFIG.group };
  // 兼容旧数据：数字→布尔
  if (typeof config.group?.show === 'number') config.group.show = config.group.show === 1;
  // 校验
  if (!['updown', 'time'].includes(config.setting?.sortType)) config.setting.sortType = 'updown';
  if (!['optional', 'position'].includes(config.group?.type)) config.group.type = 'optional';
  return config;
}
```
