# S17 — 权限安全与健康诊断
> 阶段：Phase 6 | 前置依赖：S01 | 来源：v2.0 §10 + §3.13 + 附录C

## 1. 概述

本模块涵盖三个方面：Tauri 应用权限与安全配置（权限声明、CSP 白名单、Token 密钥链存储、输入校验）、健康诊断指标采集与 Command 暴露、云同步接口（Worker URL + 微信扫码认证 + 密钥链存储）。这些内容确保应用运行安全、可观测、可同步。

### 4.3 Token 密钥链存储

```rust
use keyring::Entry;

/// 将 JWT Token 存入系统密钥链（不写磁盘文件）
pub fn store_token(token: &str) -> Result<()> {
    let entry = Entry::new("tickpulse", "auth-token")?;
    entry.set_password(token)?;
    Ok(())
}

/// 从系统密钥链读取 JWT Token
pub fn get_token() -> Result<String> {
    let entry = Entry::new("tickpulse", "auth-token")?;
    let token = entry.get_password()?;
    Ok(token)
}

/// 删除系统密钥链中的 Token
pub fn delete_token() -> Result<()> {
    let entry = Entry::new("tickpulse", "auth-token")?;
    entry.delete_password()?;
    Ok(())
}
```

use regex::Regex;
use once_cell::sync::Lazy;

/// secid 格式白名单正则
/// 合法格式：市场ID.股票代码，如 "1.600519"、"0.000001"、"116.HSI"
static SECID_PATTERN: Lazy<Regex> = Lazy::new(|| {
    Regex::new(r"^\d{1,3}\.\w{1,10}$").unwrap()

/// 校验 secid 格式
pub fn validate_secid(secid: &str) -> bool {
    SECID_PATTERN.is_match(secid)

### 4.5 健康诊断指标采集

```rust
use sysinfo::System;

impl HealthMetrics {
    /// 采集当前健康指标
    pub fn collect(
        last_refresh_ms: u64,
        refresh_history: &[(i64, u64)],  // (timestamp, duration_ms)
        emit_latency_ms: u64,
        circuit_breakers: &DashMap<String, Arc<CircuitBreaker>>,
        db_path: &Path,
        start_time: Instant,
    ) -> Self {
        // 计算1分钟平均刷新耗时
        let now = chrono::Utc::now().timestamp();
        let recent: Vec<_> = refresh_history.iter()
            .filter(|(t, _)| now - *t < 60)
            .collect();
        let avg_refresh_ms = if recent.is_empty() { 0 }
            else { recent.iter().map(|(_, d)| *d).sum::<u64>() / recent.len() as u64 };

        // 断路器状态
        let circuit_breaker_status: Vec<(String, CircuitState)> = circuit_breakers.iter()
            .map(|entry| (entry.key().clone(), *entry.value().state.read().unwrap()))
            .collect();

        // 进程 RSS
        let mut sys = System::new();
        sys.refresh_processes();
        let memory_rss_mb = sys.process(sysinfo::get_current_pid().unwrap())
            .map(|p| p.memory() / 1024 / 1024)
            .unwrap_or(0);

        // SQLite 文件大小
        let db_size_mb = std::fs::metadata(db_path)
            .map(|m| m.len() / 1024 / 1024)
            .unwrap_or(0);

        // 运行时长
        let uptime_secs = start_time.elapsed().as_secs();

        Self {
            last_refresh_ms,
            avg_refresh_ms,
            emit_latency_ms,
            circuit_breaker_status,
            memory_rss_mb,
            db_size_mb,
            uptime_secs,
        }
    }
}
```

### 4.6 云同步认证流程

```
微信扫码 → 获取 token + openid → 存系统密钥链
                                    ↓
                Worker URL CRUD（Authorization: Bearer <token>）
```

**认证流程详解**：

1. 用户点击"云同步" → 前端请求微信扫码二维码
2. 用户使用微信扫码 → 获取 `token`（JWT）+ `openid`
3. `token` 存入系统密钥链（macOS Keychain / Windows Credential Manager / Linux Secret Service）
4. `openid` 作为用户唯一标识，关联云端数据
5. 后续 CRUD 请求携带 `Authorization: Bearer <token>` 头
6. Token 过期时重新扫码认证

**Worker URL**：

```rust
const WORKER_URL: &str = "https://noisy-disk-b11f.yxbzcy.workers.dev/";
```

**CRUD 接口**：

| 操作 | 方法 | 路径 | 说明 |
|------|------|------|------|
| 读取 | GET | /sync/data | 获取云端数据 |
| 写入 | PUT | /sync/data | 推送本地数据到云端 |
| 删除 | DELETE | /sync/data | 删除云端数据 |

所有请求需携带 `Authorization: Bearer <token>` 头。

### 4.7 安全措施速查

| 措施 | 说明 | 实现位置 |
|------|------|----------|
| CSP 白名单 | 限制前端只能连接指定域名 | tauri.conf.json |
| 禁用 eval | 禁止 eval() 和 Function() | CSP + Vite 配置 |
| Token 密钥链 | JWT 存系统密钥链，不写磁盘 | keyring crate |
| secid 正则校验 | 白名单正则过滤非法输入 | Rust Command 入口 |
| 权限最小化 | 仅声明必要权限 | permissions JSON |
| 窗口白名单 | 仅 main/suspend 窗口有权限 | windows 数组 |
