# 📋 配置示例

此目录包含 TickPulse 的配置示例文件。

## 文件说明

| 文件 | 说明 |
|-----|------|
| `llm-config.example.json` | LLM API 配置示例 |
| `alert-rules.example.json` | 预警规则配置示例 |

## 使用方法

1. 复制示例文件到应用配置目录：
   - Windows: `%APPDATA%/TickPulse/config/`
   - macOS: `~/Library/Application Support/TickPulse/config/`
   - Linux: `~/.config/TickPulse/config/`

2. 根据实际情况修改配置内容

3. 重启应用使配置生效
